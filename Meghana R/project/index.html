Sentiment Analysis using Logistic Regression

Prompt

Classify text into Positive or Negative sentiment using Machine Learning.

Code

import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression

# DATASET
data = {
    'text': [
        # Positive
        'good product',
        'very good',
        'excellent product',
        'nice and useful',
        'very useful',
        'useful product',
        'happy and satisfied',
        'this is good',
        'works well',
        'amazing product',
        'good and useful',
        'really good',
        'great quality',
        'best product',

        # Negative
        'bad product',
        'very bad',
        'worst experience',
        'not good',
        'not useful',
        'waste of money',
        'terrible product',
        'does not work',
        'very bad quality',
        'poor product',
        'worst item',
        'not worth'
    ],
    'label': [1]*14 + [0]*12
}

df = pd.DataFrame(data)

# TF-IDF (handles "not good")
vectorizer = TfidfVectorizer(ngram_range=(1,2))
X = vectorizer.fit_transform(df['text'])
y = df['label']

# Model
model = LogisticRegression(max_iter=500)
model.fit(X, y)

# TEST
tests = [
    "this product is good",
    "very useful product",
    "worst item"
]

for t in tests:
    vec = vectorizer.transform([t])
    pred = model.predict(vec)[0]

    if pred == 1:
        print(t, "-> Positive")
    else:
        print(t, "-> Negative")

Results

The model successfully predicts sentiment. Example outputs:

  • This product is good → Positive
  • This product is not good → Negative

Inference

Using TF-IDF with n-grams improves model accuracy and helps handle negation like "not good".
