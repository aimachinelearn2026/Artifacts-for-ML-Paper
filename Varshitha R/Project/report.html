Phishing Website Detection using Advanced Machine Learning Techniques

1. Project Overview

This project focuses on detecting phishing websites using advanced Machine Learning techniques. Phishing attacks are one of the most dangerous cybersecurity threats where attackers create fake websites to steal sensitive information such as usernames, passwords, and banking credentials.

The system classifies websites as:

  • Legitimate Website
  • Phishing Website

The original research paper used only the K-Nearest Neighbor (KNN) algorithm with an accuracy of 85.08%. This extended work improves the original paper using multiple Machine Learning models, feature scaling, ensemble learning, hyperparameter tuning, PCA, ROC analysis, and real-time phishing prediction.

2. Objectives

  • Detect phishing websites automatically using Machine Learning
  • Improve phishing detection accuracy using advanced algorithms
  • Compare multiple Machine Learning models
  • Implement ensemble learning techniques
  • Optimize KNN using GridSearchCV
  • Perform feature scaling and dimensionality reduction
  • Visualize phishing dataset relationships using heatmaps and graphs
  • Build a real-time phishing prediction system

3. Dataset Description

The phishing dataset contains URL and website-related features used for classification.

Feature Description
URL_Length Length of the URL
having_IP_Address Checks whether URL contains IP address
SSLfinal_State Checks HTTPS security state
