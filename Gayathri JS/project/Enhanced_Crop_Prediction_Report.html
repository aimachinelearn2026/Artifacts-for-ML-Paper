03
Preprocessing
Label Encoding · Deduplication · Scaling · Train/Test Split
CodeRemove duplicates
df.drop_duplicates(inplace=True)
CodeEncode target labels
encoder = LabelEncoder()
df['label'] = encoder.fit_transform(df['label'])
CodeFeature / target split and scaling
X = df.drop('label', axis=1)
y = df['label']

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.2, random_state=42
)
🔍 Inference
  • LabelEncoder converts the 22 crop string labels to integers 0–21, required for XGBoost and the ANN's softmax output.
  • StandardScaler normalizes features to zero-mean / unit-variance, critical for distance-based (KNN) and gradient-based (LR, ANN) models.
  • An 80/20 split yields 1,760 training and 440 test samples — sufficient for reliable generalization estimates given the balanced classes.

06
Artificial Neural Network (ANN)
Keras Sequential · 128→64→32→22 · 100 epochs
CodeBuild and train ANN
ann = Sequential()
ann.add(Dense(128, activation='relu', input_dim=X_train.shape[1]))
ann.add(Dense(64, activation='relu'))
ann.add(Dense(32, activation='relu'))
ann.add(Dense(22, activation='softmax'))

ann.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy']
)

history = ann.fit(
    X_train, y_train,
    epochs=100,
    batch_size=32,
    validation_split=0.2
)
