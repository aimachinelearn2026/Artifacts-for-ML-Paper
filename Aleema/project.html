1. Problem Statement

The objective of this project is to predict whether a patient has heart disease based on medical attributes such as age, cholesterol, and blood pressure using Artificial Neural Networks (ANN). Early prediction helps in preventing serious health conditions.

2. Dataset

  • Dataset: Heart Disease Dataset (CSV)
  • Features: age, sex, cp, trestbps, chol, etc.
  • Target: 0 (No Disease), 1 (Disease)

3. Methodology

  • Data Collection (Heart dataset)
  • Data Preprocessing (cleaning and scaling)
  • Feature Scaling using StandardScaler
  • Train-Test Split (80% training, 20% testing using stratified sampling)
  • Model Training using ANN
  • Model Evaluation using accuracy, confusion matrix, and classification report

4. Code

import pandas as pd

5. ANN Model Architecture

  • Input Layer: All features
  • Hidden Layer 1: 16 neurons (ReLU)
  • Hidden Layer 2: 8 neurons (ReLU)
  • Output Layer: 1 neuron (Sigmoid)
  • Optimizer: Adam
  • Loss Function: Binary Crossentropy
