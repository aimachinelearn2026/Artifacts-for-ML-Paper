04 / Insights

Key Takeaways

Observations from training and evaluation across all five machine learning models.

🏆Winner: Logistic Regression

Despite being the simplest model, Logistic Regression achieved 92.9% accuracy. Feature scaling and class balancing were critical to its success, suggesting the decision boundary between trend/no-trend is largely linear in feature space.

⚡ANN Underperformance

The Artificial Neural Network (10 hidden neurons, single layer) scored only 51% — barely above chance. With 3,500 training samples and limited depth, it couldn't generalize. Deeper architectures and larger datasets would likely improve this significantly.

🌿Feature Correlations

Gaussian Naive Bayes (50.4%) exposed a key dataset characteristic: stock market features like MA_7, MA_30, and MOMENTUM are highly correlated. The independence assumption inherent to Naive Bayes is fundamentally violated here.

📈Target Balance

1,800 "No Trend" vs 1,700 "Trend" samples — a near-perfect 51/49 split. This balanced class distribution simplified training and means accuracy is a reliable metric, unlike in typical imbalanced financial datasets.

🎯
Production Recommendation

Deploy Logistic Regression as the primary model (92.9% accuracy) with Decision Tree as a secondary interpretable alternative (85.6%). The Decision Tree's rule structure makes it valuable for explainability and compliance use cases. Consider ensemble methods (Random Forest, Gradient Boosting) as a next iteration to potentially surpass 95%.
