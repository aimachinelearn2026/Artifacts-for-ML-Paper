Phishing Website Detection using Machine Learning

Cybersecurity Project using KNN and Logistic Regression

1. Introduction

Phishing is one of the most dangerous cybersecurity attacks where attackers create fake websites that look similar to legitimate websites in order to steal sensitive information such as usernames, passwords, and banking details.

As internet usage increases, phishing attacks are becoming more advanced and difficult to detect. Traditional detection systems such as blacklist-based methods fail to identify newly created phishing websites, also known as zero-day attacks.

Machine Learning provides an intelligent solution to this problem by learning patterns from data and automatically detecting malicious websites.

2. Problem Statement

Traditional phishing detection systems rely on manually maintained rules and blacklists. These approaches fail when attackers continuously modify URLs and website structures.

Some major challenges include:

  • Detecting zero-day phishing attacks
  • Handling dynamically changing URLs
  • Reducing false positives
  • Automatically classifying websites in real time

Therefore, there is a need for an intelligent system capable of learning patterns and accurately identifying phishing websites.

3. Research Paper Reference

This project is inspired by the research paper:

“Phishing Websites Classification using Hybrid SVM and KNN Approach”

The paper proposes a hybrid model combining KNN and SVM algorithms for phishing website classification. The model achieved around 90% accuracy.

4. Dataset Description

The dataset used in this project contains multiple features related to website behavior and URL structure. These features help machine learning models distinguish between phishing and legitimate websites.

Important Features:

  • URL Length
  • HTTPS Usage
  • Domain Age
  • Website Traffic
  • Use of Special Characters
  • Redirecting URLs

The dataset is divided into:

  • 80% Training Data
  • 20% Testing Data

5. Machine Learning Algorithms

K-Nearest Neighbors (KNN)

KNN is a supervised learning algorithm that classifies data points based on similarity. It identifies the nearest neighboring points and assigns the majority class.

Logistic Regression

Logistic Regression predicts the probability of a website being phishing or legitimate. It is efficient and widely used for classification problems.

6. Implementation

Python and Scikit-learn libraries were used to implement the machine learning models. The dataset was loaded, preprocessed, and divided into training and testing sets.

import pandas as pd

from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

data = pd.read_csv("phishing.csv")

X = data.drop("class", axis=1)
y = data["class"]

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

knn = KNeighborsClassifier(n_neighbors=5)
knn.fit(X_train, y_train)

lr = LogisticRegression(max_iter=1000)
lr.fit(X_train, y_train)
