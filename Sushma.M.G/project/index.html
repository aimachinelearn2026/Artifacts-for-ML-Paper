Classification Algorithms Theory - KNN

  • KNN is a supervised classification algorithm
  • Proposed by Evelyn Fix & Joseph Hodges (1951)
  • Further developed by Thomas Cover (1967)
  • Stores all training data (no explicit training phase)
  • Classifies data based on similarity (distance)
  • Uses k nearest neighbors to decide the class
  • Common distance metric: Euclidean distance
  • Works on majority voting principle
  • Simple and easy to implement
  • Used in classification and regression problems

Algorithm 1: KNN Pseudo-code

Input:
x, l, s
x : training data
l : label
s : sample to classify

For each training data point:
    Compute distance d(xi, s)

Select desired number k of nearest neighbors
Sort distances in increasing order
Count occurrences of labels among top k neighbors

Output:
Assign to s the most frequent label

Linear Discriminant Analysis (LDA)

  • LDA is a supervised learning algorithm
  • Introduced by Ronald Fisher (1936)
  • Used for dimensionality reduction and classification
  • Projects data into lower-dimensional space
  • Maximizes between-class variance
  • Minimizes within-class variance
  • Improves class separation

LDA Feature Space Transformation

  • Analyzes entire feature space
  • Creates new axes in lower-dimensional space
  • Reduces 2D data (X₁, X₂) to 1D
  • Projects data onto optimal axis
  • Maximizes between-class variance
  • Minimizes within-class variance
  • Achieves better class separability

Simple Perceptron

  • Introduced by Rosenblatt
  • Basic unit of Artificial Neural Networks (ANN)
  • Inspired by biological neuron
  • Used as supervised binary classifier
  • Classifies data into two classes (0 or 1)
  • Input
  • Weight and bias
  • Net sum
  • Activation function
