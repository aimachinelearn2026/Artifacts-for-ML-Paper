Intrusion Detection System using Machine Learning on NSL-KDD Dataset

Reference:Department of CSE, Jaypee Institute of Information Technology, Sector 62, NOIDA, India

Problem statement: The objective of this project is to detect network intrusions using machine learning models and classify network traffic as Normal or Attack using the NSL-KDD dataset.

Data description: NSL-KDD dataset is an improved version of KDD99 dataset used for intrusion detection. It contains network traffic data with normal and attack classes.

Processing steps: Steps:

  1. Load dataset
  2. Handle categorical features
  3. Normalize data using MinMaxScaler
  4. Feature selection using Chi-square
  5. Split into training and testing sets

Inference: KNN: K-Nearest Neighbors achieved good accuracy by classifying data based on similarity between instances. However, it has higher computational cost during testing since it stores all training data and computes distance for each prediction. SVM: Support Vector Machine performed well by finding an optimal hyperplane that separates normal and attack traffic. It provides good generalization but requires more training time for large datasets. Random Forest: Random Forest performed efficiently by combining multiple decision trees. It reduces overfitting and provides high stability and accuracy in intrusion detection tasks. Naive Bayes : Naive Bayes assumes independence between features and works well with simple probabilistic modeling. However, its performance is lower due to unrealistic feature independence assumptions. Logistic Regression: Logistic Regression provides probabilistic classification using a linear decision boundary. It performs reasonably well but struggles with complex nonlinear relationships in the dataset. Decision Tree: Decision Tree achieved the highest accuracy among all models. It works by splitting data based on feature conditions and handles nonlinear relationships effectively, making it suitable for intrusion detection.

models = {
    "KNN": KNeighborsClassifier(),
    "SVM": SVC(),
    "Random Forest": RandomForestClassifier(n_estimators=200),
    "Naive Bayes": GaussianNB(),
    "Logistic Regression": LogisticRegression(max_iter=1000),
    "Decision Tree": DecisionTreeClassifier()
}

results = {}

for name, model in models.items():
    model.fit(X_train, y_train)
    y_pred = model.predict(X_test)

    acc = accuracy_score(y_test, y_pred)
    results[name] = acc

    print(f"\n{name} Accuracy: {acc:.4f}")
