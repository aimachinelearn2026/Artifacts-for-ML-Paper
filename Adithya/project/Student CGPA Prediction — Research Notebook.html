Section 2 Data Generation

🧪 Synthetic Dataset Generation

Since the original research dataset is not publicly available, a synthetic dataset of 500 students is generated based on the paper's feature structure. The target final_CGPA is derived from pre_CGPA with penalties for failed and forced-withdrawal courses.

[2]
data
▶ 
np.random.seed(42)
n = 500

data = pd.DataFrame({
    "age":                   np.random.randint(22, 40, n),
    "gender":               np.random.choice(["Male", "Female"], n),
    "nationality":          np.random.choice(["Indian", "Other"], n),
    "marital_status":       np.random.choice(["Single", "Married"], n),
    "job_status":           np.random.choice(["Working", "Not Working"], n),
    "residence":            np.random.choice(["Urban", "Rural"], n),
    "bachelor_university":  np.random.choice(["Govt", "Private"], n),
    "bachelor_major":       np.random.choice(["Science", "Health", "Other"], n),
    "pre_CGPA":             np.round(np.random.uniform(2.0, 4.0, n), 2),
    "english_score":        np.random.randint(50, 100, n),
    "program":              np.random.choice(["Diploma", "Masters", "PhD"], n),
    "semester":             np.random.randint(1, 6, n),
    "complementary_courses":np.random.randint(0, 3, n),
    "avg_courses_per_sem":  np.random.randint(2, 6, n),
    "total_courses":        np.random.randint(10, 30, n),
    "failed_courses":       np.random.randint(0, 5, n),
    "withdrawal_courses":   np.random.randint(0, 3, n),
    "forced_withdrawal":    np.random.randint(0, 2, n),
    "credit_hours":         np.random.randint(30, 120, n),
    "department":           np.random.choice(["Epidemiology", "Biostatistics"], n),
    "specialization":       np.random.choice(["Public Health", "General"], n),
})

# Target variable — logic from research paper
data["final_CGPA"] = (
    data["pre_CGPA"]
    - (data["failed_courses"] * 0.3)
    - (data["forced_withdrawal"] * 0.2)
    + np.random.normal(0, 0.2, n)
)
data["final_CGPA"] = data["final_CGPA"].clip(0, 4)
data.to_csv("student_performance.csv", index=False)

Section 4 Preprocessing

🔧 Preprocessing & Feature Engineering

Categorical variables are encoded via one-hot encoding. CGPA is categorized into 3 classes for classification tasks. Features are scaled using StandardScaler.

[7]
preprocessing
▶ 
# CGPA → class labels for classification
def categorize(cgpa):
    if cgpa >= 3.3:   return 2   # High
    elif cgpa >= 2.0: return 1   # Medium
    else:              return 0   # Low

data["cgpa_class"] = data["final_CGPA"].apply(categorize)

# One-hot encode categorical columns
data = pd.get_dummies(data, drop_first=True)

# Feature/target split
X_reg = data.drop(["final_CGPA", "cgpa_class"], axis=1)
y_reg = data["final_CGPA"]

X_clf = data.drop(["final_CGPA", "cgpa_class"], axis=1)
y_clf = data["cgpa_class"]

# Train/test split
X_train_r, X_test_r, y_train_r, y_test_r = train_test_split(
    X_reg, y_reg, test_size=0.2, random_state=42)
X_train_c, X_test_c, y_train_c, y_test_c = train_test_split(
    X_clf, y_clf, test_size=0.2, random_state=42)

Section 7 Ensemble Model

🤝 Ensemble Model (VotingRegressor)

A VotingRegressor combines Random Forest and Linear Regression to produce a blended prediction, leveraging the strengths of both models.

rf_reg = RandomForestRegressor()

    ("lr", lin)
])

print("Ensemble MAE:", mean_absolute_error(y_test_r, pred_ens))
