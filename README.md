# Student Artifacts Repository

This repository contains the submitted artifacts and resources prepared by students as part of their extended resource paper work.

Each student has an individual folder named after them, containing their respective project materials and supporting files.

## Contents

The repository may include:

- Presentation files related to the project or paper
- Extended research papers formatted in IEEE style
- HTML-based project/code demonstrations
- Supporting project artifacts and documentation

---
Maintained for academic and documentation purposes.
