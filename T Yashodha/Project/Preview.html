📋 Project Introduction

This machine learning project recommends the most suitable crop to grow based on soil and climate conditions. The model analyzes Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH, and Rainfall to predict the optimal crop from 22 possible categories.

The notebook was developed and executed in Google Colab using Python, Pandas, Seaborn, and Scikit-learn.

Cell 6 Train-Test Split

Cell 7 Model Training — Random Forest

model = RandomForestClassifier() model.fit(X_train, y_train)
RandomForestClassifier trained successfully. Estimators : 100 decision trees (default) Criterion : gini impurity Max features: sqrt(n_features)

Insights

Random Forest uses 100 decision trees by default for robust ensemble predictions.
Each tree is trained on a random bootstrap sample of the training data.
Gini impurity measures how often a randomly chosen element is misclassified.
Ensemble voting makes it resilient to individual tree overfitting.

Cell 8 Model Accuracy

y_pred = model.predict(X_test) accuracy = accuracy_score(y_test, y_pred) print('Accuracy:', accuracy)
