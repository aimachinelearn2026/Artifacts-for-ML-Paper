K-Data Mining Concept

  • Definition: KNN is a simple non-parametric classification algorithm.
  • Basic Idea: Classify based on nearest neighbors.
  • Distance Calculation: Uses Euclidean distance.
  • Choosing K: Select K closest data points.
  • Voting: Majority voting decides class.

Mathematical Model of KNN

Feature Vector: x = (x1, x2, x3, ..., xn)

Distance Formula:

d(x,y) = √ Σ (xi - yi)²

KNN Research Method

  • Instance-based learning
  • Lazy learning method
  • K = 1 → Nearest Neighbor
  • High accuracy with large data
